# 用户手册

## 安装

### Windows

1. 下载 `WinCapture-1.0.0.exe`
2. 双击安装
3. 按向导完成安装

### macOS

1. 下载 `WinCapture-1.0.0.dmg`
2. 拖动到Applications
3. 首次运行需授权屏幕录制权限

### Linux

```bash
# 下载AppImage
chmod +x WinCapture-1.0.0.AppImage
./WinCapture-1.0.0.AppImage
```

## 首次配置

首次启动自动弹出配置窗口：

| 字段 | 说明 | 示例 |
|------|------|------|
| 服务器地址 | Kimiclaw DB地址 | `https://api.company.com` |
| 用户ID | 你的员工ID | `zhangsan` |
| API Key | 从管理员获取 | `key_abc123` |

点击保存后，程序最小化到系统托盘。

## 日常使用

### 托盘图标操作

右键点击托盘图标：

- **状态**：查看当前采集状态
- **暂停/恢复**：临时停止/恢复采集
- **配置**：修改设置
- **退出**：关闭程序

### 采集状态

图标颜色含义：

| 颜色 | 状态 |
|------|------|
| 🟢 绿色 | 正常运行 |
| 🟡 黄色 | 已暂停 |
| 🔴 红色 | 连接失败 |

## 隐私控制

### 暂停采集

临时需要隐私时：
1. 右键托盘图标
2. 选择"暂停采集"
3. 图标变为黄色

恢复时选择"恢复采集"。

### 应用白名单

默认只采集工作应用：
- ✅ VSCode、IDEA等开发工具
- ✅ Chrome、Edge等工作浏览器
- ✅ Outlook、邮件客户端
- ❌ 微信、QQ等社交软件
- ❌ 游戏、视频等娱乐软件

## 常见问题

### Q: 为什么有些应用没有被采集？
A: 非工作应用被过滤了。如需采集，请联系管理员添加白名单。

### Q: 采集的图片会被其他人看到吗？
A: 不会。图片加密存储，只有你自己和管理员能查看。

### Q: 可以删除历史数据吗？
A: 可以。联系管理员或发送邮件到 support@kimiclaw.com

### Q: 程序会影响电脑性能吗？
A: 影响很小。CPU占用<5%，内存<200MB。

### Q: 离线时数据会丢失吗？
A: 不会。数据本地缓存，联网后自动上传。
