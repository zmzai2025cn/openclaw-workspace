# 代码审查清单

提交代码前，请逐项检查：

## 编译前检查

- [ ] 代码能在本地编译通过
- [ ] 无编译警告（或已确认可忽略）
- [ ] 所有 using 语句已排序（System 在前）

## 代码质量检查

- [ ] 所有 public 方法有 XML 文档注释
- [ ] 复杂逻辑有行内注释
- [ ] 无魔法数字（使用常量）
- [ ] 字符串使用 `nameof` 而非硬编码

## 异常处理检查

- [ ] 所有外部调用有 try-catch
- [ ] 异常被记录到日志（非 Debug.WriteLine）
- [ ] 资源使用 using 或 try-finally 释放
- [ ] 无空 catch 块

## Null 安全检奁

- [ ] 参数检查使用 `?? throw`
- [ ] 可空类型使用 `?` 标记
- [ ] 属性访问使用 `?.` 和 `??`
- [ ] 返回值检查 null

## 性能检查

- [ ] 无字符串拼接循环（使用 StringBuilder）
- [ ] 大对象使用池化或复用
- [ ] 异步方法使用 async/await

## 安全检奁

- [ ] 无硬编码密码/密钥
- [ ] 文件路径使用 Path.Combine
- [ ] 用户输入已验证

## 测试检查

- [ ] 新功能有对应测试
- [ ] 所有测试通过
- [ ] 边界条件已测试

## 文档检查

- [ ] CHANGELOG 已更新
- [ ] README 已更新（如需要）
- [ ] 版本号已更新

---

## 快速自检命令

```bash
# 编译
dotnet build

# 格式检查
dotnet-format --check

# 测试
dotnet test
```
