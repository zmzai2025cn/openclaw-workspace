# 变更日志 (Changelog)

所有 notable 变更都记录在此文件。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

---

## [0.1.0] - 2026-02-22

### 添加
- 初始验证版本发布
- P2P 网络层：基于 libp2p 的节点发现与连接
- 消息协议：JSON 格式的消息编解码
- 任务路由：支持本地执行和远程委托
- OpenClaw 集成：通过子进程调用 agent
- CLI 界面：交互式命令行
- 文档：README、架构设计、部署指南、API 文档

### 技术栈
- Node.js >= 18.0.0
- libp2p 1.x
- OpenClaw >= 2026.2.9

### 已知限制
- 仅支持局域网或公网直连（无 NAT 穿透）
- 简单命令解析（不支持复杂引号嵌套）
- 无持久化存储
- 单线程执行
